# Contribution
If you want to contribute there are a few things to note:
1. The use of AI to help in any way is prohibited
2. You should use Twine with the story format of SugarCube (This is for consistency)
3. You must ensure that the information is accurate
4. You must specify the mod version
5. Only SFW mods are allowed
